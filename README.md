# eic-bochka

### Installation
First clone the repository.

git clone https://github.com/JeffersonLab/eic-bochka.git

cd eic-bochka


### Compilation
./gradlew clean 

./gradlew bochka



### Running
cd bin
./bochka <path to the yaml file>

Note update running script to point the correct location of the ec-bochka installation.
